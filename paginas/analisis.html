INTRODUCCION AL DESARROLLO DE SOFTWARE

En el proceso de analisis se reunen con el cliente para identificar las metas, se analiza las perspectivas del cliente, evaluar los conceptos que tiene el cliente para establecer su viabilidad, las necesidades y requerimientos, realizar un analisis tecnico y economico, asignar funciones al Hardware, Software, personal, base de datos y otro elementos del sistema.

Analisis de Requisitos: se debe hacer reconocimiento del problema, Evaluar y sintetizar, Modelado de la posible solucion, Especifiación y Revisión; el proyecto tendra varias modifiaciónes durante la identificación de las necesidades.

ESTUDIO DE VIABILIDAD

la viabilidad y el analisis de resgos estan ligados de muchas maneras, en caso de que el riesgo del proyecto sea alto, la viabilidad de producir software de calidad se reduce!!!
